

# MedAl:Healthcare Portal Web Application

## Overview
MedAl is a digital healthcare solution designed to reduce overcrowding and long queues in South African public hospitals. The platform provides patients with an online portal to schedule appointments, perform virtual check-ins, and receive medication reminders improving healthcare efficiency and patient satisfaction.

## Problem Statement
South African public healthcare facilities face severe overcrowding due to:
- Increasing demand from an ageing population and rising chronic diseases
- Inefficient appointment scheduling and patient flow systems
- Poor communication between patients and hospital staff

MedAl addresses these challenges by moving healthcare management online, reducing the number of patients physically present in facilities.

## Features
- Online appointment scheduling with doctors
- Virtual check-in system — patients submit basic information before arriving
- Symptoms checker with guidance and doctor contact details
- Medication reminders to improve adherence and reduce missed doses
- Hospital resource availability checker for nearby facilities
- Staff dashboard for managing appointments and monitoring queues in real time

## Technologies Used
- HTML
- CSS
- JavaScript

## Lessons Learned
- User experience is critical — every design decision must keep the end user in mind
- Simplicity is key — the platform must be accessible to patients of all ages and tech abilities

## Challenges
- Could not implement a real-time chat feature due to resource constraints
- Ensuring design consistency and ease of navigation across all pages

## Project Status
⚠️ This is a prototype developed as an academic project. Some planned features were not fully implemented due to resource limitations.
